# **App Name**: Readex

## Core Features:

- Intelligent Capture Center: Log multi-session consumption of books, articles, and videos with granular status tracking and detailed session logs.
- Belief & Mental Model Vault: Explicitly define personal principles, life rules, and worldviews using a structured data model in a database for high-confidence distillation.
- Insight Distillation Tool: AI-powered tool that reasons through reading notes and annotations to extract declarative core claims for your belief vault.
- Evolutionary Timeline: A database-driven log that records how, when, and why your principles shift, documenting your intellectual growth over time.
- Concept Atlas UI: An interactive concept-mapping interface to visually arrange and link research nodes using custom spatial layouts.
- Drafting Atelier: A writing workspace that features a contextual sidebar to pull evidence, beliefs, and quotes directly from the Library while you write.
- Reflective Goal Tracker: Manage and persist sophisticated reading goals by source type with a real-time progress dashboard powered by a Firestore database.

## Style Guidelines:

- Primary color: Scholastic Ink (#2D1F33), a deep, rich violet-tinted charcoal that evokes intellectual authority and deep focus.
- Background color: Ivory Manuscript (#F7F6F8), a soft, high-light paper tone desaturated from the primary ink for extreme legibility.
- Accent color: Academic Blue (#2C47C4), an analogous hue used for calls to action, highlights, and active node indicators.
- Headline font: 'Playfair Display' for an elegant, high-contrast, editorial feel suited for philosophy and long-form literature. Body font: 'Alegreya' to provide an intellectual and warm serif reading experience.
- Monospaced font: 'Source Code Pro' used specifically for metadata, counters, and technical timestamps.
- Dual-pane layouts with persistent context panels that ensure research evidence is always visible alongside active work surfaces.
- Quiet, serif-style transitions using vertical drifts to mimic the slow, thoughtful page-turning of a scholar's journal.
